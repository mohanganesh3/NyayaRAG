# NyayaRAG Launch Checklist

## Comparison Demo Workflow

1. Pick one high-signal practitioner question with binding authority.
2. Run the same prompt across NyayaRAG and comparison tools.
3. Verify every cited authority against the underlying source before publishing.
4. Capture the point where NyayaRAG downgrades uncertainty instead of asserting false confidence.

## Benchmark Story Sequence

1. Lead with the trust number and the measured benchmark route.
2. Explain citation fabrication versus misgrounding in plain legal language.
3. Show the update timestamp and benchmark metadata before comparison claims.
4. Close on filing risk: a lawyer should not need a second tool just to verify the first one.

## Initial Distribution Checklist

- Publish one comparison demo clip with the trust page linked in the caption.
- Prepare chamber-shareable screenshots for WhatsApp and LinkedIn.
- Track contacted chambers, institutions, and practitioner groups in a live follow-up sheet.
- Keep every launch asset tied back to measurable trust, not abstract AI claims.
